Хроматический образ (`Chromatic Image <https://2e.aonprd.com/Spells.aspx?ID=882>`_) / Закл. 6
""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

- иллюзия
- визуальный

**Обычай**: арканный, оккультный

**Сотворение**: |д-2| жестовый, словесный

**Продолжительность**: 1 минута

**Источник**: Secrets of Magic pg. 95

----------

Вы вызываете 3 своих иллюзорных образа, и каждый из вас четверых получает различный цвет, который меняется каждый раз, когда кто-то атакует вас.
Это имеет эффекты :ref:`spell--m--Mirror-Image`, но когда противник уничтожает один из образов, киньте 1d4, чтобы узнать какого цвета был образ на момент уничтожения.
Атакующий получает соответствующий эффект.

+----+-----------+-----------------------------+
| d4 |    Цвет   |            Эффект           |
+====+===========+=============================+
|  1 | Красный   | 5 урона огнем и             |
|    |           | 5 ментального урона         |
+----+-----------+-----------------------------+
|  2 | Оранжевый | 5 урона кислотой и          |
|    |           | 5 ментального урона         |
+----+-----------+-----------------------------+
|  3 | Желтый    | 5 урона электричеством и    |
|    |           | 5 ментального урона         |
+----+-----------+-----------------------------+
|  4 | Зеленый   | 5 урона ядом и              |
|    |           | 5 ментального урона         |
|    |           | + :c_frightened:`напуган 1` |
+----+-----------+-----------------------------+

Хроматический доспех (`Chromatic Armor <https://2e.aonprd.com/Spells.aspx?ID=881>`_) / Закл. 4
""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

- преграждение
- свет

**Обычай**: арканный, оккультный

**Сотворение**: |д-2| или |д-3| жестовый, словесный

**Дистанция**: касание

**Цели**: 1 готовое существо

**Продолжительность**: 1 минута

**Источник**: Secrets of Magic pg. 95

----------

Вы окутываете цель доспехом из листов цветного света.
Доспех излучает яркий свет в 20 футах (и тусклый свет в следующих 20 футах).
Всякий раз, когда существо атакует цель и находится рядом с ней, то атакующий должен совершить спасбросок Воли в конце своего действия.
При провале, он :c_dazzled:`ослеплен` до конца своего следующего хода.
Вне зависимости от результата своего спасброска, атакующий временно иммунен до конца своего следующего хода.
Ослепляющий эффект имеет признаки :t_light:`свет` и :t_visual:`визуальный`.

Когда вы сотворяете это заклинание, дважды киньте 1d8 по таблице далее, чтобы определить цвета доспеха (перекидывая повторы).
Каждый цвет дает сопротивление 5 обозначенному типу урона.
Если вы тратите 3 действия на :ref:`Сотворение (Cast) <action--Cast-a-Spell>` этого заклинания, то киньте 1d8 трижды.

+----+---------------+-----------------------------------------------------------+
| d8 |      Цвет     |                         Тип урона                         |
+====+===============+===========================================================+
|  1 | Красный       | огонь                                                     |
+----+---------------+-----------------------------------------------------------+
|  2 | Оранжевый     | кислота                                                   |
+----+---------------+-----------------------------------------------------------+
|  3 | Желтый        | электричество                                             |
+----+---------------+-----------------------------------------------------------+
|  4 | Зеленый       | яд                                                        |
+----+---------------+-----------------------------------------------------------+
|  5 | Синий         | звук                                                      |
+----+---------------+-----------------------------------------------------------+
|  6 | Индиго        | ментальный                                                |
+----+---------------+-----------------------------------------------------------+
|  7 | Фиолетовый    | сила                                                      |
+----+---------------+-----------------------------------------------------------+
|  8 | | Совпадающий | | Доспех соответствует цвету, последнего типа урона,      |
|    | | цвет        | | полученного целью за прошедшую минуту.                  |
|    |               | | Если она не получала никакого из этих 7 типов урона     |
|    |               | | или доспех уже такого цвета, то перекиньте этот бросок, |
|    |               | | а также дальнейшие результаты с 8.                      |
+----+---------------+-----------------------------------------------------------+

----------

**Усиление (+3)**: Сопротивление увеличивается на 5.
